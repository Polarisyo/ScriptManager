# AI视频分镜剧本管理系统 技术方案

---

## 一、系统概述

### 1.1 项目背景
AI视频分镜剧本管理系统是一个集剧本管理、AI智能生成、素材管理于一体的创作平台，旨在帮助视频创作者高效完成从文本剧本到可视化分镜的全流程创作。

### 1.2 目标用户
* 独立视频创作者
* 短视频制作团队
* 广告创意策划
* 教育培训内容制作人
* 个人兴趣爱好者

### 1.3 核心价值
* **降低创作门槛**：通过AI能力辅助非专业创作者
* **提升创作效率**：自动化重复性分镜创作工作
* **保证创作质量**：提供专业级的创作模板和工具
* **可视化创作过程**：实时预览创作成果

## 三、核心功能模块

### 3.1 用户管理模块
*   **功能描述**：负责用户注册、登录、个人信息管理及权限控制。
*   **功能清单**：
    *   用户注册：邮箱/手机号注册，验证码验证
    *   用户登录：账号密码登录，JWT令牌认证
    *   个人信息管理：头像、昵称、简介修改
    *   账户设置：密码修改、通知偏好设置
    *   订阅管理：免费/付费套餐选择和管理
    *   积分系统：AI使用积分管理
*   **关键接口**：
    *   `POST /api/v1/auth/register` - 用户注册
    *   `POST /api/v1/auth/login` - 用户登录
    *   `POST /api/v1/auth/logout` - 用户登出
    *   `GET  /api/v1/user/profile` - 获取用户信息
    *   `PUT  /api/v1/user/profile` - 更新用户信息
    *   `GET  /api/v1/user/subscription` - 获取订阅信息
    *   `POST /api/v1/user/subscription/upgrade` - 升级订阅
    *   `GET  /api/v1/user/credits` - 获取积分余额

### 3.2 项目管理模块
*   **功能描述**：管理用户创建的AI视频项目，包括项目CRUD、版本管理。
*   **功能清单**：
    *   项目创建：新建项目，设置项目基本信息
    *   项目列表：按状态、时间筛选项目
    *   项目详情：查看项目完整信息
    *   项目编辑：修改项目标题、描述、设置
    *   项目删除：软删除项目，可恢复
    *   项目复制：复制已有项目作为模板
    *   版本管理：创建项目版本快照，支持版本回滚
    *   项目统计：统计项目镜头数、完成进度等
*   **关键接口**：
    *   `POST /api/v1/projects` - 创建项目
    *   `GET  /api/v1/projects` - 获取项目列表
    *   `GET  /api/v1/projects/{id}` - 获取项目详情
    *   `PUT  /api/v1/projects/{id}` - 更新项目
    *   `DELETE /api/v1/projects/{id}` - 删除项目
    *   `POST /api/v1/projects/{id}/duplicate` - 复制项目
    *   `GET  /api/v1/projects/{id}/versions` - 获取版本列表
    *   `POST /api/v1/projects/{id}/versions` - 创建版本
    *   `POST /api/v1/projects/{id}/restore/{version}` - 恢复版本

### 3.3 剧本管理模块
*   **功能描述**：处理剧本的导入、编辑、解析和AI增强。
*   **功能清单**：
    *   剧本导入：支持文本粘贴、文件上传
    *   剧本编辑：在线文本编辑器，支持格式标记
    *   剧本解析：自动识别场景、角色、台词
    *   AI剧本拆分：一键将完整剧本拆分为分镜结构
    *   剧本优化：AI润色剧本，提升语言质量
    *   剧本版本：记录剧本修改历史
    *   剧本导出：导出为 PDF/Word 格式
*   **关键接口**：
    *   `POST /api/v1/projects/{id}/script/upload` - 上传剧本
    *   `PUT  /api/v1/projects/{id}/script` - 更新剧本
    *   `GET  /api/v1/projects/{id}/script` - 获取剧本
    *   `POST /api/v1/ai/parse-script` - AI 解析剧本
    *   `POST /api/v1/ai/enhance-script` - AI 优化剧本
    *   `GET  /api/v1/projects/{id}/script/versions` - 剧本版本历史
    *   `POST /api/v1/projects/{id}/script/export` - 导出剧本

### 3.4 分镜管理模块
*   **功能描述**：管理视频分镜的创建、编辑、排序和状态跟踪。
*   **功能清单**：
    *   分镜创建：手动创建或AI批量生成分镜
    *   分镜列表：树状目录展示所有分镜
    *   分镜详情：查看分镜完整信息，包括剧本、素材、草稿
    *   分镜编辑：修改分镜标题、描述、时长等
    *   分镜排序：拖拽调整分镜顺序
    *   分镜复制：复制分镜结构和内容
    *   分镜状态管理：标记分镜完成状态
    *   分镜批量操作：批量删除、状态更新
*   **关键接口**：
    *   `POST /api/v1/projects/{id}/shots` - 创建分镜（批量）
    *   `GET  /api/v1/projects/{id}/shots` - 获取分镜列表
    *   `GET  /api/v1/shots/{id}` - 获取分镜详情
    *   `PUT  /api/v1/shots/{id}` - 更新分镜
    *   `DELETE /api/v1/shots/{id}` - 删除分镜
    *   `POST /api/v1/shots/{id}/duplicate` - 复制分镜
    *   `PUT  /api/v1/projects/{id}/shots/reorder` - 重新排序
    *   `POST /api/v1/shots/batch-update` - 批量更新

### 3.5 素材管理模块
*   **功能描述**：管理分镜相关的图片、视频、音频等素材资源。
*   **功能清单**：
    *   AI生成素材：根据分镜描述生成图片素材
    *   素材上传：支持多格式文件上传（图片、视频、音频）
    *   素材管理：查看、编辑、删除素材
    *   素材预览：在线预览图片/视频
    *   素材搜索：按标签、类型、时间搜索素材
    *   素材分类：自定义素材标签和分类
    *   素材重用：复制素材到其他分镜
    *   素材处理：图片压缩、格式转换、视频截图
*   **关键接口**：
    *   `POST /api/v1/shots/{shotId}/materials` - 上传素材
    *   `GET  /api/v1/shots/{shotId}/materials` - 获取素材列表
    *   `GET  /api/v1/materials/{id}` - 获取素材详情
    *   `PUT  /api/v1/materials/{id}` - 更新素材信息
    *   `DELETE /api/v1/materials/{id}` - 删除素材
    *   `POST /api/v1/ai/generate-image` - AI生成图片
    *   `POST /api/v1/materials/search` - 搜索素材
    *   `POST /api/v1/materials/{id}/copy` - 复制素材

### 3.6 视频草稿模块
*   **功能描述**：基于分镜和素材生成视频草稿，支持预览和编辑。
*   **功能清单**：
    *   AI生成视频：基于分镜和素材生成视频片段
    *   草稿列表：查看分镜的所有视频草稿版本
    *   草稿预览：在线播放视频草稿
    *   草稿下载：下载视频文件
    *   草稿重新生成：使用新参数重新生成视频
    *   草稿版本管理：管理不同版本的视频草稿
    *   草稿参数设置：设置视频分辨率、帧率、时长
    *   视频导出：导出最终视频文件
*   **关键接口**：
    *   `POST /api/v1/shots/{shotId}/drafts` - 生成视频草稿
    *   `GET  /api/v1/shots/{shotId}/drafts` - 获取草稿列表
    *   `GET  /api/v1/drafts/{id}` - 获取草稿详情
    *   `DELETE /api/v1/drafts/{id}` - 删除草稿
    *   `POST /api/v1/drafts/{id}/regenerate` - 重新生成
    *   `GET  /api/v1/drafts/{id}/download` - 下载视频
    *   `POST /api/v1/drafts/{id}/export` - 导出视频

        *   DALL-E
    *   **文生视频模型**
        *   Runway Gen-2
        *   Stable Video Diffusion
        *   Pika Labs
*   **关键接口**：
    *   `POST /api/v1/ai/generate/shots` - AI生成分镜
    *   `POST /api/v1/ai/enhance/script` - AI优化剧本
    *   `POST /api/v1/ai/generate/image` - AI生成图片
    *   `POST /api/v1/ai/generate/video` - AI生成视频
    *   `POST /api/v1/ai/analyze/continuity` - 检查连贯性
    *   `POST /api/v1/ai/analyze/rhythm` - 分析节奏

*   **关键接口**：
    *   `POST /api/v1/tasks` - 创建任务
    *   `GET  /api/v1/tasks/{taskId}` - 获取任务状态
    *   `GET  /api/v1/tasks` - 获取任务列表
    *   `DELETE /api/v1/tasks/{taskId}` - 取消任务
    *   `POST /api/v1/tasks/{taskId}/retry` - 重试任务

### 4.1 性能需求
*   **页面加载**：首页加载时间 < 3秒

*   **故障恢复**：关键服务故障恢复时间 < 5分钟

*   **数据加密**：敏感数据加密存储，HTTPS传输

*   **防攻击**：防止SQL注入、XSS、CSRF攻击

### 4.4 可扩展性需求
*   **水平扩展**：支持无状态服务水平扩展
*   **模块化**：微服务架构，支持独立部署
*   **API版本**：支持API版本管理，向后兼容

## 五、数据流程设计

### 5.1 核心业务流程
#### 5.1.1 新建项目流程
1.  用户登录系统
2.  点击“新建项目”
3.  填写项目基本信息（标题、描述、类型）
4.  选择创建方式：
    *   a. 空白项目：创建空项目，后续手动添加
    *   b. 导入剧本：上传或粘贴剧本文本
5.  系统创建项目并跳转到项目详情页

#### 5.1.2 AI生成分镜流程
1.  在项目详情页点击“AI生成分镜”
2.  选择生成参数（风格、详细程度等）
3.  提交生成任务，系统返回任务ID
4.  前端通过WebSocket监听任务进度
5.  AI服务处理任务：
    *   a. 解析剧本结构
    *   b. 识别场景转换点
    *   c. 生成分镜描述
    *   d. 估算每个分镜时长
6.  生成完成后，前端刷新分镜列表

#### 5.1.3 生成视频草稿流程
1.  在分镜详情页点击“生成视频草稿”
2.  选择素材和生成参数
3.  提交视频生成任务
4.  AI服务处理：
    *   a. 组合选定的图片素材
    *   b. 根据描述生成视频过渡效果
    *   c. 添加基础音效（如可用）
    *   d. 合成视频文件
5.  生成完成后提供预览和下载

## 六、关键设计决策

### 6.1 架构设计决策
*   **微服务架构**：考虑到AI服务可能独立升级，采用微服务便于独立部署。
*   **前后端分离**：前端使用Vue3+TS，后端提供RESTful API。
*   **异步处理**：AI生成任务使用消息队列，避免阻塞用户请求。
*   **多存储方案**：结构化数据用MySQL，非结构化用MongoDB，文件用OSS。

### 6.2 AI集成决策
*   **多模型支持**：不绑定单一AI服务商，支持切换和扩展。
*   **成本控制**：积分系统控制AI使用成本，防止滥用。
*   **结果缓存**：AI生成结果缓存，相同输入避免重复计算。
*   **降级策略**：AI服务不可用时提供基础功能。

### 6.3 用户体验决策
*   **实时反馈**：AI生成任务提供进度条和实时状态。
*   **版本控制**：所有关键操作支持撤销和版本回滚。
*   **模板系统**：提供常用视频类型的创作模板。
*   **离线支持**：基础编辑功能支持离线操作。
